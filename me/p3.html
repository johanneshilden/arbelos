Liquid Epsilon

Haskell Mobile-first compile-to-javascript

Cross-platform technologies and solutions are special interests of mine. This project provides a smorgasbord of modules to assist in building single-page, client-side web and mobile applications using Haskell and the UHC compiler.

Features

  • Easy JSON parsing and serialization
  • Local storage persistence of Haskell data
  • Hashtag routing using convenient do-notation
  • DOM manipulation API
  • Functional reactive programming
  • WebSocket client API

Example

ul <- createElement "ul"
    setAttribute ul "id" "top-menu"
    sequence $ map (flip (~>) ul) $ reverse $ map snd es
    return ul

LessEdit

C++ Qt Markdown

A WYSIWYG Markdown editor for desktop environments, written in C++ and based on the Sundown parser and Qt's WebKit rendering engine.

En desktop-applikation som översätter Markdown till HTML i realtid och som även kan exportera till PDF och andra format. LessEdit är utvecklad i C++ med hjälp av ramverket Qt. Lösningen fungerar således lika bra på Windows och Mac OS som under Linux samt andra X11-system.
