RESTful APIs for relational data

The Trombone server simplifies adaptation of relational data to mobile-friendly APIs operating within the RESTful web service paradigm, with data exchange driven by JSON. It enables one-to-one mapping of HTTP requests to database statements according to rules layed out by a set of route templates.

# Create a new customer
#   e.g. POST /customer { "name"     : "ACME", 
#                         "phone"    : "555-BIZNESS", 
#                         "industry" : "Robotics" }
POST /customer  <>
  INSERT INTO customers (name, phone, industry)
    VALUES ({{name}}, {{phone}}, {{industry}})
